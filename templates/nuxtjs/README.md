This is a [Nuxt](https://nuxt.com/) project generated by the Babylon.js Editor.

## Getting Started

Install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

The Babylon.js scene is rendered on the client (the project is configured with `ssr: false`).
You can start editing the page by modifying `src/app.vue`.
